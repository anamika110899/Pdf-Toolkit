#  PDF Toolkit

A modern full-stack web application to manage PDF files easily and efficiently.

---

##  Project Overview

**PDF Toolkit** is a web-based application designed to simplify working with PDF documents.  
It provides essential tools such as merging, splitting, compressing, converting, and organizing PDF files through a clean and user-friendly interface.

The project focuses on performance, security, and responsiveness, making it suitable for real-world use.

---

##  Features

- Merge multiple PDF files into one
- Split PDF into separate pages
- Compress large PDF files
- Convert PDF to Word and Word to PDF
- OCR for scanned PDF documents
- Upload and manage files securely
- Responsive modern UI
- Fast and optimized processing

---

##  Tech Stack

### Frontend
- React.js
- Material UI (MUI)
- Framer Motion
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- Multer (File Upload)
- PDF Processing Libraries
- Tesseract OCR
